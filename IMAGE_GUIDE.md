# 초대장 이미지 분할 가이드

이 가이드는 첨부하신 초대장 이미지를 최적으로 분할하는 방법을 설명합니다.

## 필요한 이미지 파일 (총 4개)

```
images/
├── envelope-back.png      # 봉투 하단 본체 (직사각형)
├── envelope-flap.png      # 봉투 상단 뚜껑 (직사각형)
├── wax-seal.png          # 실링 왁스 (원형, 투명배경)
└── decorations.png       # 장식 요소들 (선택사항)
```

---

## 1. envelope-back.png (봉투 하단부)

### 크롭 영역:
- **전체 이미지에서 하단 직사각형 부분만**
- 가로선(실링 왁스 위치) 아래 부분
- 크기: 약 1200px × 650px (비율 유지)

### 포함 요소:
✅ 하단 장식 (눈송이, 나뭇잎)
✅ 좌우 모서리 장식
✅ "INVITATION" 텍스트는 **제외** (CSS로 처리)

### Photoshop 작업:
1. 이미지 열기
2. 사각형 선택 도구로 하단부 선택
3. 가로선(중앙 라인) 바로 아래부터 하단 끝까지 선택
4. 이미지 → 자르기
5. 파일 → 내보내기 → PNG로 저장
6. 파일명: `envelope-back.png`

---

## 2. envelope-flap.png (봉투 뚜껑 - 직사각형)

### 크롭 영역:
- **상단 직사각형 부분** (실링 왁스 제외)
- 가로선(중앙 라인) 위 부분 전체
- 크기: 약 1200px × 380px

### 포함 요소:
✅ 상단 직사각형 영역
✅ "INVITATION" 텍스트
✅ 상단 장식 (눈송이, 나뭇잎)
✅ 좌우 상단 모서리 장식
❌ 실링 왁스는 **제외** (별도 파일로)

### 중요: 배경 포함!

이 부분은 투명 배경이 **아닙니다**. 검은 배경과 금색 장식을 모두 포함한 직사각형입니다.

### Photoshop 작업:
1. 이미지 열기
2. 사각형 선택 도구로 상단부 선택
3. 최상단부터 가로선(중앙 라인) 바로 위까지 선택
4. 실링 왁스가 있는 중앙선은 **포함하지 않음**
5. 이미지 → 자르기
6. 파일 → 내보내기 → PNG로 저장
7. 파일명: `envelope-flap.png`

### 예상 모양:
```
┌─────────────────────────┐
│   ❄  ❄  INVITATION  ❄  ❄   │
│  🌿               🌿  │
│   (상단 장식들 포함)     │
└─────────────────────────┘
      (직사각형 뚜껑)

* 이 부분이 CSS로 회전하면서
  위로 열리는 애니메이션
```

## 3. wax-seal.png (실링 왁스)

### 크롭 영역:
- **중앙의 금색 실링 왁스만**
- 크기: 약 200px × 200px (정사각형)

### 포함 요소:
✅ 금색 원형 실링
✅ "RIME" 텍스트
✅ 실링의 질감과 그림자
❌ 주변 배경은 **완전 투명**

### 중요: 고품질 PNG!

실링 왁스는 클릭 가능한 버튼이므로 고해상도가 중요합니다.

### Photoshop 작업:
1. 타원 선택 도구로 실링 왁스 원형 선택
2. 여유있게 선택 (약간의 그림자 포함)
3. 선택 → 수정 → 페더 (5px 정도)
4. 레이어 → 새로 만들기 → 복사된 레이어
5. 배경 레이어 삭제
6. 파일 → 내보내기 → PNG로 저장 (투명도 유지)
7. 파일명: `wax-seal.png`

### 예상 모양:
```
    ◯◯◯◯◯
  ◯        ◯
 ◯   RIME   ◯
  ◯        ◯
    ◯◯◯◯◯
(원형 금색 실링)
```

## 4. decorations.png (선택사항 - 추가 장식)

개별 장식 요소를 따로 추출하고 싶다면:

### 분리 가능한 요소:
- 모서리 장식 (4개)
- 눈송이 (여러 개)
- 나뭇잎 장식
- 별 모양

각각을 별도 PNG 파일로 저장하면 애니메이션 시 개별 제어가 가능합니다.

## 이미지 저장 설정

### PNG 내보내기 옵션:
- **형식**: PNG-24
- **투명도**: 활성화 ✓
- **인터레이스**: 없음
- **색상 프로파일**: sRGB
- **품질**: 최상 (100%)

### 권장 크기:
- envelope-back.png: 1200 × 650px
- envelope-flap.png: 1200 × 400px
- wax-seal.png: 300 × 300px (고해상도)

## 온라인 도구로 분할하기 (Photoshop 없을 경우)

### 1. Remove.bg (배경 제거)
https://www.remove.bg/
- 실링 왁스 배경 투명화에 사용

### 2. Photopea (무료 온라인 포토샵)
https://www.photopea.com/
- Photoshop과 동일한 방식으로 작업 가능
- PSD 파일 지원

### 3. Canva (간단한 크롭)
https://www.canva.com/
- 이미지 자르기와 배경 제거 기능
