# AI 이미지 편집 프롬프트 (구글 나노 바나나용)

첨부된 초대장 이미지를 3개로 분할하기 위한 프롬프트입니다.

---

## 🖼️ 작업 1: envelope-back.png (봉투 하단 본체)

### 프롬프트:

```
이미지의 하단 부분만 남기고 크롭해주세요.

- 중앙의 가로 금색 라인(실링 왁스가 있는 선) 바로 아래부터 이미지 맨 끝까지
- 실링 왁스는 포함하지 마세요
- 하단 장식(눈송이, 나뭇잎, 모서리 장식)은 모두 포함
- 배경 검은색 유지
- 직사각형 형태로 자르기
```

### 간단 버전:

```
중앙 금색 라인 아래부터 맨 아래까지만 잘라주세요. 실링 왁스는 제외.
```

### 저장:

- 파일명: `envelope-back.png`
- 크기: 가로 1200px 정도 유지

## 🖼️ 작업 2: envelope-flap.png (봉투 상단 뚜껑)

```

- "INVITATION" 텍스트 포함
- 상단 장식(눈송이, 나뭇잎, 모서리 장식)은 모두 포함
- 배경 검은색 유지
- 직사각형 형태로 자르기
```

### 간단 버전:

```
맨 위부터 중앙 금색 라인 위까지만 잘라주세요. 실링 왁스는 제외.
```

### 저장:

- 파일명: `envelope-flap.png`
- 크기: 가로 1200px 정도 유지

## 🖼️ 작업 3: wax-seal.png (실링 왁스)

### 프롬프트:

```
중앙의 금색 원형 실링 왁스만 오려내주세요.

- 중앙의 금색 원형 도장만 선택
- "RIME" 텍스트 포함
- 원형으로 크롭
- 주변 배경은 완전히 투명하게 제거
- 실링의 그림자와 광택 효과는 유지
```

### 간단 버전:

```
중앙의 금색 원형 실링만 잘라내고 배경은 투명하게 만들어주세요.
```

### 저장:

- 파일명: `wax-seal.png`
- 크기: 300 x 300px (정사각형)
- **중요**: PNG 형식, 배경 투명

## 📋 전체 작업 순서

1. **첫 번째**: 하단 부분 크롭 → `envelope-back.png`
2. **두 번째**: 상단 부분 크롭 → `envelope-flap.png`
3. **세 번째**: 실링 왁스만 추출 (배경 투명) → `wax-seal.png`
